The hackathon

In order to develop the PCBWriter into a device that is easy for others to use and copy, we plan a hackathon

December 14th/15th, 2013
at Dingfabrik, Cologne

to which we would like to invite everyone interested. Please register at pcbwriter@dingfabrik.de. Food and drinks will be provided.

Begin: December 14th, 2013, 11:00
End: December 15th, 2013, 17:00

See also: http://dingfabrik.de/events/420/pcbwriter-hackathon/.

Some ideas for projects:

  • Control polygon mirror assemblies from various laser printers
  • User-friendly host software (with GUI?)
  • Mechanical fixes for existing device
  • Experiment with double-layer boards
  • CNC drilling, automatic alignment with OpenCV
  • Improved documentation
  • ...
